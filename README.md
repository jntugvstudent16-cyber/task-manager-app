## 📝 About The Project
A beautiful, simple task manager built entirely on GitHub for Hack & Forge 2026. Keep track of your daily tasks with ease!

## ✨ Features
- ✅ Add new tasks
- ✅ Mark tasks as complete
- ✅ Delete individual tasks
- ✅ Clear all tasks
- ✅ Filter tasks (All/Pending/Completed)
- ✅ Statistics dashboard
- ✅ Data saved in browser
- ✅ Works on all devices

## 🛠️ Built With
- **HTML5** - Structure
- **CSS3** - Styling & Animations
- **JavaScript** - Functionality
- **GitHub Pages** - Hosting
- **LocalStorage** - Data persistence

## 🎯 How To Use
1. Type a task in the input box
2. Click "Add Task" or press Enter
3. Check the box to mark as complete
4. Use filters to view different tasks
5. Click × to delete a task
6. Use "Clear All" to remove all tasks
